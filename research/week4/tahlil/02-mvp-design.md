# MVP Dizayn va Arxitektura (1-2 oy)

**Stack:** Telegram Bot API + Landing Page (Front) | Python/Node.js (Back) | PostgreSQL (DB) | Redis (Cache) | Click/Uzum to'lov

**Arxitektura:** Telegram Bot ↔ Backend API ↔ PostgreSQL | Landing Page + Admin Panel

**MVP v1.0 funksiyalar:**
- Telegram Bot: /start (registratsiya), /new (listing), /search, /my (dashboard)
- Web: Listinglarni ko'rish, profil
- Backend: User/Listering CRUD, Category, Search API

**Screen flow:**
```
/start → Registratsiya
/new → Kategoriya → Mahsulot → Narx → Rasm → Tasdiqlash
/search → Filter → Listing → Xabar (chat)
/my → Listinglarim → Sotilgan → Stat
```
