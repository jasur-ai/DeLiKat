# Mom Test — Suhbat Metodologiyasi

**Manba:** Rob Fitzpatrick | **Asosiy muammo:** Odamlar sizni xursand qilish uchun "ha" deyishadi

> Agar suhbatda odam sizga yolg'on gapirsa ("ha, yaxshi fikr" desa) — bu suhbat foydasiz.

---

## Yolg'on pozitiv signallari

| Signal | Haqiqat | Detection |
|--------|---------|-----------|
| "Ajoyib fikr!" | Sizni xursand qilmoqchi | "Hozir nima qilyapsiz?" |
| "Men buni ishlatardim" | Hech qachon ishlatmaydi | "Oxirgi marta qachon bo'lgan?" |
| "Bu muammo bor" | Kichik muammo | "Qancha pul sarflagansiz?" |
| "Pul to'lashga tayyor" | Hech qachon to'lamaydi | "Byudjetingizda bormi?" |
| Davomiyligi 5 min | Muloyimlik | 30+ min = real qiziqish |
| General javoblar | Kichik muammo | Detail = real muammo |
| Faqat javob beradi | Muloyimlik | O'zi savol bersa = real |

## Yomon vs Yaxshi savollar

| Yomon (yolg'on) | Yaxshi (real) |
|----------------|---------------|
| "Agar platforma bo'lsa, ishlatarmidingiz?" | "Oxirgi 3 oyda sotilmay qolgan mahsulot bilan nima qildingiz?" |
| "Bu yaxshi fikrmi?" | "Qancha pul yo'qotdingiz? Aniq summa?" |
| "Bu muammo muhimmi?" | "Buni hal qilish uchun nima qilib ko'rgansiz?" |
| "Qancha to'lagan bo'lar edingiz?" | "Hozir qanday yechimdan foydalanayapsiz?" |

## Commitment scale

| Daraja | Misol | Signal |
|--------|-------|--------|
| 0 | "O'ylab ko'raman" | ❌ Hech narsa |
| 1-2 | "Testni sinab ko'raman" | 🟢 Past |
| 3-4 | "Yana gaplashamiz / ofisga keling" | 🟡 O'rta |
| 5-6 | "5 ta tadbirkorni tavsiya qilaman / listing beraman" | 🔴 Yuqori |
| 7 | "Pul to'lashga tayyorman" | 🔴💰 Eng yuqori |

**Agar commitment 0-1 bo'lsa — yolg'on pozitiv. 3+ bo'lsa — real muammo.**
